## 🚀 Getting Started

Once you launch the app, you see a menu. The menu allows you to choose two different windows or image files. Click the Open button in the top corner to load your first file. Click the second Open button to load the file you want to compare. The software places both items in the main viewing area. You use your mouse to move the images or windows. A slider tool allows you to fade between the two choices. This highlights gaps or shifts in your layout instantly.

## 🎛️ Using the Interface

The interface stays simple. You have three main controls:

- Load: Brings your files into the workspace.
- Compare: Triggers the overlay mode to show differences.
- Reset: Clears the workspace for a new task.

You can hold your left mouse button to drag items around. Use your scroll wheel to zoom in or zoom out. This allows you to check very small details on a screen. If you zoom in tight, the software keeps the views aligned. This prevents the two images from drifting apart.

## 📋 Common Tasks

Use these steps for your daily work:

- Checking buttons: Load a mockup and the live site. Overlap them to see if the button size matches.
- Font validation: Load a design file and the actual window output. Check if the text matches the weight and spacing.
- Alignment check: Use the grid toggle tool to draw lines over your project. This ensures items sit in the correct spots.

## 🆘 Troubleshooting

If the software does not open, check these items:

- Ensure your Windows version updates to the latest release.
- Shut down other heavy programs to free up system resources.
- Run the installer as an administrator by right-clicking the file.
- Verify that your antivirus does not flag the app. Some security tools mark new files as unknown. You may need to click "More Info" and then "Run Anyway" if this happens.

We build this app to be fast and light. It does not run background tasks when you close the main window. This keeps your computer performance steady. If you encounter a window that does not load, try closing the app and starting it again. This resets the visual engine and clears the cache files.
